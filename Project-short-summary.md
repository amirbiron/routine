# Routine Anchors – שגרה בחוסר שגרה

**ריפו:** [amirbiron/routine](https://github.com/amirbiron/routine)
**סטטוס:** פעיל

כלי אינטראקטיבי לבניית עוגני שגרה יומיים לילדים בתקופות של חוסר ודאות – ממשק עשיר עם Drag-and-Drop ו-Push notifications.

## פיצ'רים מרכזיים
- בניית שגרה יומית אינטראקטיבית עם Drag-and-Drop
- אימות משתמשים עם JWT ו-Push notifications
- ממשק RTL מלא עם Radix UI, Framer Motion ו-Dark mode
- גרפים ואנליטיקה (Recharts)
- Full-stack type-safety עם tRPC
- ניהול אדמין

## Tech Stack
TypeScript, React 19, Vite, Tailwind CSS, Express, tRPC, Drizzle ORM, MySQL, JWT, Radix UI, Framer Motion, pnpm
